## 🧮 CÁLCULO DEL ÍNDICE PANAS

### Fórmula Base

$$
\text{PANAS}_{Lipo}(t) = \frac{\text{Precio Lipo}(t)}{\text{Precio Lipo}(2010)}
$$

### Ejemplos de Cálculo

#### 2015
$$
\text{PANAS}_{2015} = \frac{2,941}{1,714} = 1.715
$$

#### 2020
$$
\text{PANAS}_{2020} = \frac{4,462}{1,714} = 2.603
$$

#### 2025
$$
\text{PANAS}_{2025} = \frac{5,385}{1,714} = 3.140
```

### Interpretación

- **PANAS = 1.00** en 2010 (año base)
- **PANAS = 3.14** en 2025
- **Significado**: El poder adquisitivo se multiplicó por 3.14 en 15 años

## 📊 PROYECCIONES FUTURAS

### Modelo de Crecimiento

Utilizamos un modelo de crecimiento logístico para proyectar el futuro:

$$
P(t) = P_0 \times e^{rt} \times \frac{1}{1 + \frac{P_0}{K}(e^{rt} - 1)}
$$

Donde:
- $P_0$ = Precio inicial (2010)
- $r$ = Tasa de crecimiento
- $K$ = Capacidad máxima del mercado
- $t$ = Tiempo en años

### Escenarios de Proyección

#### Escenario Conservador (r = 0.04)

def validate_lipo_price(price, year):
    # Validar rango de precios
    if price < 1000 or price > 20000:
        raise ValueError("Precio fuera de rango esperado")
    
    # Validar crecimiento anual
    if year > 2010:
        prev_price = get_previous_price(year - 1)
        growth_rate = (price - prev_price) / prev_price
        if growth_rate > 0.5 or growth_rate < -0.2:
            raise ValueError("Crecimiento anómalo")
    
    return True
